# Règles (Drools)

Place ici tes `.drl` (ex.: `src/main/resources` dans un projet Quarkus Kogito).
Dans ce squelette, le volume `./rules-service/rules` est monté dans le conteneur.
Adapte l’image du `rules-service` à ton projet Kogito qui charge ces règles.
